The Cloud Resume Challenge

Cloud Resume Challenge Completed (November, 2022)

I completed the Cloud Resume Challenge, which involved creating a website with HTML and CSS and hosting it on S3 as a static site. I configured a DNS hosted zone along with the purchase of a domain name and added a CloudFront Distribution to it.

The project requires a webpage visitor counter application that is deployed with a serverless architecture using AWS Lambda running a Python script, as well as a DynamoDB table and an API Gateway which triggers the Lambda function.

The webpage has JavaScript on it to hit the REST API when the page is loaded. This architecture was provisioned using an infrastructure as code deployment with a YAML template and CloudFormation. The AWS Serverless Application Model and the Linux command line interface was used to deploy the code to AWS and build a CI/CD pipeline. Virtualization for Linux was achieved using Hashicorp Vagrant to deploy a Ubuntu Linux virtual machine.

A GitHub Actions CI/CD pipeline was set up to implement version control for the back-end code stored in a GitHub repository.
